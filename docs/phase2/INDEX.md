## Phase 2 Feature Summary

- **Billing System** — Track court appearances → auto-generate GST invoices
- **Advocate Portal** — Role hierarchy (Solicitor → AOR → Junior → Clerk → Client)
- **Law Firm Accounts** — Shared matter management, fee splitting
- **Order Tracking** — Daily court order PDF scraping + OCR
- **ECODE Normalization** — Normalized section headers (65% DB storage reduction)
- **AI Layer** — Hearing date prediction, order outcome extraction

None of Phase 2 is required for Phase 1 launch.
